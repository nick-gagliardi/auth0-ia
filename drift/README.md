# Docs Drift Watcher

This folder contains **weekly snapshots** of `index/index.json` and a **weekly IA drift report** generated by `node scripts/drift-report.mjs`.

- Snapshots: `drift/snapshots/YYYY-MM-DD/index.json`
- Reports: `drift/reports/YYYY-MM-DD.md` and `drift/reports/latest.md`

These artifacts are intended to be published by the scheduled GitHub Action (`.github/workflows/docs-drift-watcher.yml`) to the `drift-reports` branch.
